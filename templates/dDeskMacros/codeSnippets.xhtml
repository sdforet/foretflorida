Load-Time Helper

Can be used to grab the load-time for a section of code

<?php
//some helpful XDEBUG functions
var_export();
echo xdebug_print_function_stack($var);
//use together to get the load time of the contained code
$debugStart = number_format(xdebug_time_index(), 5);
//code to test for time
$debugFinish = number_format(xdebug_time_index(), 5);
echo  "LOADTIME = " . number_format(($debugFinish-$debugStart), 5);

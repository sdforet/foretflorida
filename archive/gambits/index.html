Introduction

Sean's Gambits is a site which is being developed for academic work at Santa Fe College where I am a current student. The original exercise work from the course was a web safe bright blue background color, with light blue text, and page elements had a 2px light blue border around them. I have redesigned the site, with the approval of my professor, to this more appropriate game site.


The majority of the content can be found in the Play Games link to the left. The first several games were accomplished using the exacting specifications given in our text, however begining with the Game of 21 (which I am currently redeveloping) and continuing with Tic Tac Toe and HangMan, the games have been dramatically enhanced from the original exercise parameters. Please feel free to browse through this site and just a note, two of these games are in my profile site with links to the script files, be sure to check those out on the Code Examples tab of my profile site.
